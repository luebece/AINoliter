🎮 플레이 방법

- 로그인 불필요: 모든 서비스는 로그인이 필요없으니 그냥 클릭하고 플레이 하시면 됩니다

💬 채팅 기능

- 모델 선택: 2가지 AI 모델 중 선택 가능하며, 모델 변경 시 이전 대화 기록은 초기화됩니다.

🕹️ 게임 기능

- 햄버거 메뉴: 모델 선택, 선택지 횟수 설정(기본 10회), 대화 기록 확인 가능

- 선택지 횟수: 실제 플레이 횟수 = 설정값 + 2 (게임 시작/이름 입력 포함)

- 추천 설정: 짧게 10회, 보통 20회, 장시간 50회

- 참고: 로딩이 오래 걸릴 경우(30초 이상) 이는 api 오류이니, 선택지 버튼을 다시 한번 클릭

🤖 AI 대화

- 설정 방법: 모델1과 모델2의 성격을 각각 설정

- 진행 방식: 대화 시작/대화 계속 버튼 클릭 시 모델1 → 모델2 순차적 대화 진행

📖 AI 노벨

- 장르 선택: 초기 장르 선택 시 AI가 스토리 시작

- 자유 모드: 장르 미선택 시 일반 AI 채팅으로 사용 가능

- 작동 방식: 사용자 입력 → AI 스토리 생성 반복
